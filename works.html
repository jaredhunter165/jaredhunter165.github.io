Android Microphone

Consisting of one Android application and a Java desktop application, the project utilizes an android device as a microphone in an auditorium like setting by sending audio packets over a wireless network using a UDP connection. A common use case would be in a conference like setting, where a moderator could pass speaking permission to attendees via the server side application without having to pass on a physical microphone. This project was made using Eclipse for the server side application and Android Studio for the Android application.
